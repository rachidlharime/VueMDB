# vuemdb

A simple VueJS 2 movies application using BootstrapVue, vue-router and vue-axios .

Movies data provided by OMDB API : http://www.omdbapi.com/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
